Manipulate Arrays w/ push()
	.push() - takes one or more parameters and adds to the end of the array
		<array_name>.push(num, "item1", ["array_item1", "array_item2"])
	
	.unshift() - adds elements to the beginning of an array


	.pop() - removes item from the end of an array; this item can be stored
			by assigning it to a variable.

		<array_name>.pop(num, "item")

		const popped_value = array.pop()

	.shift() - removes the first element

	function plusThree(num) {

	}


Scope - refers to visibility of variables. Variables defined outside
  of a function block have Global scope; they can be seen everywhere
  in your code.

	-Variables declared w/o "let" or "const" are automatically
	  created in the "global" scope so always declare with "let" or "const"
